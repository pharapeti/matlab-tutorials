This demo requires R2017b or NEWER.

To start this demo, do the following:

1.) >> bh_car_startup

2.) interact with the app to select the DEMO or REPORT of interest.

3.) Click on the HELP tab sheet of the DEMO selector and launch the
    demo GUIDE presentation for comments on what each of the demos do
    AND how to interact with them.
